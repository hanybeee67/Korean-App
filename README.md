# 에베레스트 한국어 교육 프로그램 (Everest Korean Tutor)

네팔 직원들을 위한 AI 기반 한국어 학습 웹 애플리케이션입니다.

## 기능 (Features)
- 🎧 **TTS (Text-to-Speech)**: 한국어 발음 듣기
- 🎤 **STT (Speech-to-Text)**: 한국어 말하기 연습 및 발음 평가
- 📋 **상황별 학습**: 홀, 주방, 일상 생활 등 상황에 맞는 문장 제공
- 🇳🇵 **다국어 지원**: 네팔어 발음 표기 및 뜻 제공

## 사용 방법 (Usage)
1. **[관리자]** 구글 시트에 단어를 추가하면 앱에 자동 반영됩니다.
2. **[직원]** 웹사이트 링크에 접속하여 바로 학습할 수 있습니다.


## 개발 환경 (Tech Stack)
- HTML5, CSS3 (Premium Glassmorphism Design)
- Vanilla JavaScript (No Framework)
- Web Speech API (Browser Native AI)
