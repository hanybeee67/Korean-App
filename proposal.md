# 에베레스트 레스토랑: AI 기반 한국어 교육 프로그램 제안서

## 1. 프로젝트 비전
**"내 손안의 AI 한국어 튜터"**: 60여 명의 네팔 직원들이 언제 어디서나 스마트폰으로 접속할 수 있는 교육 프로그램을 만듭니다. 단순 암기를 넘어, AI를 활용해 실제 식당 현장에서 바로 쓸 수 있는 실전 회화 능력을 키우는 것이 목표입니다.

## 2. 핵심 기능 (AI 시대의 혁신적 접근)

### A. 구글 시트 연동형 "살아있는 교본"
*   **구조**: 관리자님이 정리하신 구글 시트가 곧 앱의 데이터베이스가 됩니다.
*   **기능**: 관리자님이 시트에 새로운 단어를 추가하면, 직원들 앱에도 실시간으로 반영됩니다.
*   **구성**: 주방용, 홀 서빙용(주문, 서빙, 계산), 생활 한국어로 카테고리화하여 직무별 맞춤 학습이 가능합니다.

### B. AI 발음 코치 (오디오 중심 학습)
*   **TTS (듣기)**: 한글을 잘 못 읽는 직원을 위해 AI가 정확한 한국어 발음을 들려줍니다.
*   **STT (말하기 평가)**: 직원이 폰에 대고 한국어를 말하면, AI가 이를 듣고 한국 손님이 알아들을 수 있는지 피드백을 줍니다. (예: "발음이 정확해요!", "다시 한 번 시도해보세요.")

### C. 상황별 롤플레잉 (Roleplay) 시뮬레이션
*   **시나리오**: AI가 "한국 손님" 역할을 맡습니다. (예: "여기 김치 좀 더 주세요", "이거 너무 매워요" 등 돌발 상황 연출)
*   **상호작용**: 직원이 적절하게 응대하면 레벨 업! 실제 접객 상황에 대한 두려움을 없애줍니다.

### D. 다국어 인터페이스
*   한국어가 서툰 직원을 위해 모든 메뉴와 설명은 **네팔어**와 **영어**로 함께 제공됩니다.

## 3. 기술적 구현 방식

*   **플랫폼**: **웹 앱 (PWA)**
    *   *이유*: 앱스토어 설치 불필요. 카톡이나 문자로 링크만 보내면 바로 실행 가능. 안드로이드/아이폰 모두 호환.
*   **기술 스택**:
    *   **프레임워크**: React.js (Vite) + Tailwind CSS (모바일 최적화 디자인).
    *   **데이터**: 관리자님의 구글 시트 + PapaParse (CSV 변환).
    *   **AI 엔진**: Web Speech API (기본 듣기/말하기 무료) + Gemini API (심화 롤플레잉).

## 4. 개발 로드맵

### 1단계: 인터랙티브 교본 (기초 공사)
*   모바일 웹 UI 제작.
*   구글 시트 데이터 불러오기.
*   "터치해서 듣기" 기능 구현.

### 2단계: 실전 연습 모드
*   깜빡이(Flashcard) 암기 모드.
*   따라 말하기(Maching) 테스트.

### 3단계: AI 상황극 (고도화)
*   AI 손님과의 대화 기능 추가.

## 5. 요청 사항
*   **구글 시트 공유**: 정리해두신 구글 시트의 링크를 주시거나, 내용을 엑셀/CSV로 다운받아 업로드해 주시면 바로 적용하겠습니다.
*   **데이터 칼럼**: `분류(카테고리)`, `한국어 표현`, `네팔어 뜻`, `발음(선택사항)` 정도로 구성되어 있으면 좋습니다.
